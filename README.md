# Sistema de Gestión de Colonia Urbana

Este proyecto esta desarrollado en **Java** para gestionar una colonia urbana, incluyendo funcionalidades para el manejo de gatos y recursos. El sistema permite el seguimiento de los gatos y la administración de los recursos en los hogares de tránsito.

## Tecnologías Utilizadas

- **Java**: Lenguaje de programación principal.
- **SQL**: Para la base de datos relacional.
- **JDBC**: Conexión entre Java y la base de datos.
- **Maven**: Gestión de dependencias y construcción del proyecto.
- **JavaFX**: Para la interfaz gráfica.

## Características del Proyecto

- **Gestión de Gatos**: Permite registrar gatos callejeros, su seguimiento y su integración en hogares de tránsito.
- **Base de Datos Relacional**: Utiliza SQL para almacenar la información de los gatos, hogares de tránsito y otros recursos.
- **Interfaz Gráfica**: La aplicación tiene una interfaz intuitiva para interactuar fácilmente con el sistema.
- **Persistencia de Datos**: Se usa un archivo XML (`persistence.xml`) para gestionar la conexión y persistencia de la base de datos.
